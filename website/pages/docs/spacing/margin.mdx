# Margin

Utilities for controlling an element's margin.

| React props      | CSS Properties                                        |
| ---------------- | ----------------------------------------------------- |
| `margin=[value]` | `margin: [value];`                                    |
| `m=[value]`      | `margin: [value];`                                    |
| `mt=[value]`     | `margin-top: [value];`                                |
| `mr=[value]`     | `margin-right: [value];`                              |
| `mb=[value]`     | `margin-bottom: [value];`                             |
| `ml=[value]`     | `margin-left: [value];`                               |
| `my=[value]`     | `margin-top: [value];`<br />`margin-bottom: [value];` |
| `mx=[value]`     | `margin-right: [value];`<br />`margin-left: [value];` |

## Add margin to a single side

Control the margin on one side of an element using the `m{t|r|b|l}={size}` utilities.

For example, `mt={6}` would add `1.5rem` of margin to the top of an element, `mr={4}` would add `1rem` of margin to the right of an element, `mb={8}` would add `2rem` of margin to the bottom of an element, and `ml={2}` would add `0.5rem` of margin to the left of an element.

```jsx preview
<template preview>
  <Box display="flex" flexWrap="wrap" alignItems="flex-start" justifyContent="center" color="white" fontFamily="mono" mx={-5}>
    <Box mx={5} bg="cool-gray-200" borderRadius="md" overflow="hidden" pt={6}>
      <Box bg="rose-500" display="flex" alignItems="center" justifyContent="center" py={3} px={4}>
        mt-6
      </Box>
    </Box>
    <Box mx={5} bg="cool-gray-200" borderRadius="md" overflow="hidden" pr={4} mt={6}>
      <Box bg="rose-500" display="flex" alignItems="center" justifyContent="center" py={3} px={4}>
        mr-4
      </Box>
    </Box>
    <Box mx={5} bg="cool-gray-200" borderRadius="md" overflow="hidden" pb={8} mt={6}>
      <Box bg="rose-500" display="flex" alignItems="center" justifyContent="center" py={3} px={4}>
        mb-8
      </Box>
    </Box>
    <Box mx={5} bg="cool-gray-200" borderRadius="md" overflow="hidden" pl={2} mt={6}>
      <Box bg="rose-500" display="flex" alignItems="center" justifyContent="center" py={3} px={4}>
        ml-2
      </Box>
    </Box>
  </Box>
</template>
<Box mt={6}>mt-6</Box>
<Box mr={4}>mr-4</Box>
<Box mb={8}>mb-8</Box>
<Box ml={2}>ml-2</Box>
```

## Add horizontal margin

Control the horizontal margin of an element using the `px={size}` utilities.

```jsx preview
<template preview>
  <Box display="flex" justifyContent="center" color="white" fontFamily="mono">
    <Box bg="emerald-200" borderRadius="md" overflow="hidden" px={8}>
      <Box bg="emerald-500" py={3} px={4}>px-8</Box>
    </Box>
  </Box>
</template>
<Box px={8}>px-8</Box>
```

## Add margin to all sides

Control the margin on all sides of an element using the `p={size}` utilities.

```jsx preview
<template preview>
  <Box display="flex" justifyContent="center" color="white" fontFamily="mono">
    <Box bg="fuchsia-200" borderRadius="md" overflow="hidden" p={8}>
      <Box bg="fuchsia-500" py={3} px={4}>p-8</Box>
    </Box>
  </Box>
</template>
<Box p={8}>p-8</Box>
```

## Responsive

To control the space between elements at a specific breakpoint, use responsive object notation. For example, adding the property `py={{ md: 8 }}` to an element would apply the `py={8}` utility at medium screen sizes and above.

```jsx
<Box py={{ md: 8 }} />
```

For more information about xstyled's responsive design features, check out [Responsive Design](@TODO-link) documentation.

## Customizing

### Spacing scale

f you'd like to customize your values for space between, margin, margin, all at once, use the `theme.space` section of your theme.
