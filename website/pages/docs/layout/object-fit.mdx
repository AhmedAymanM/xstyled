# Object Fit

Utilities for controlling how a replaced element's content should be resized.

| React props         | CSS Properties         |
| ------------------- | ---------------------- |
| `objectFit=[value]` | `object-fit: [value];` |

## Contain

Resize an element's content to stay contained within its container using `objectFit="contain"`.

```jsx preview
<>
  <template preview>
    <Box bg="rose-300" borderRadius="md" overflow="hidden">
      <Box
        as="img"
        h={48}
        w={1}
        objectFit="contain"
        src="https://images.unsplash.com/photo-1490079027102-cd08f2308c73?ixlib=rb-1.2.1&ixid=MXwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHw%3D&auto=format&fit=crop&w=285&h=285&q=80"
        alt=""
      />
    </Box>
  </template>
  <Box bg="rose-300">
    <Box as="img" objectFit="contain" h={48} w="100%" />
  </Box>
</>
```

## Responsive

To control how a replaced element's content should be resized only at a specific breakpoint, use responsive object notation. For example, adding the property `objectFit={{ md: "contain" }}` to an element would apply the `objectFit="contain"` utility at medium screen sizes and above.

```jsx
<Box objectFit={{ md: 'contain' }} />
```
