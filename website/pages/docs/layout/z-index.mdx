# Z-Index

Utilities for controlling the stack order of an element.

| React props      | CSS Properties      |
| ---------------- | ------------------- |
| `zIndex=[value]` | `z-index: [value];` |

## Usage

Control the stack order (or three-dimensional positioning) of an element in xstyled, regardless of order it has been displayed, using the `zIndex={index}` utilities.

## Responsive

To apply a visibility utility only at a specific breakpoint, use responsive object notation. For example, adding the property `visiblity={{ md: "hidden" }}` to an element would apply the `visiblity="hidden"` utility at medium screen sizes and above.

```jsx
<Box visiblity={{ md: 'hidden' }} />
```

For more information about xstyled's responsive design features, check out [Responsive Design](@TODO-link) documentation.

## Customizing

### Z-Indices

If you'd like to customize your values for `zIndex`, use the `theme.zIndices` section of your theme.

```diffjs
  // theme.js
  export const theme = {
    zIndices: {
+     dialog: 200,
+     tooltip: 300,
    },
  }
```
