# Display

Utilities for controlling the display box type of an element.

| React props       | CSS Properties      |
| ----------------- | ------------------- |
| `display=[value]` | `display: [value];` |

## Block

Use `display="block"` to create a block-level element.

```jsx preview
<>
  <template preview>
    <Box p={4} spaceY={4} bg="light-blue-300" borderRadius="md">
      <Box
        as="span"
        display="block"
        borderRadius="md"
        color="white"
        fontWeight="extrabold"
        textAlign="center"
        bg="light-blue-500"
        p={6}
      >
        1
      </Box>
      <Box
        as="span"
        display="block"
        borderRadius="md"
        color="white"
        fontWeight="extrabold"
        textAlign="center"
        bg="light-blue-500"
        p={6}
      >
        2
      </Box>
      <Box
        as="span"
        display="block"
        borderRadius="md"
        color="white"
        fontWeight="extrabold"
        textAlign="center"
        bg="light-blue-500"
        p={6}
      >
        3
      </Box>
    </Box>
  </template>
  <Box spaceY={4}>
    <Box as="span" display="block">
      1
    </Box>
    <Box as="span" display="block">
      2
    </Box>
    <Box as="span" display="block">
      3
    </Box>
  </Box>
</>
```

## Flow-Root

Use `display="flow-root"` to create a block-level element with its own [block formatting context](https://developer.mozilla.org/en-US/docs/Web/Guide/CSS/Block_formatting_context).

```jsx preview

    <Box spaceY={4}>
      <Box display="flow-root" bg="purple-300" borderRadius="md">
        <Box
          my={4}
          color="white"

          textAlign="center"

</>
```

## Grid

Use `display="grid"` to create a grid container.

```jsx preview
<>
  <template preview>
    <Box
      display="grid"
      p={4}
      bg="fuchsia-300"
      borderRadius="md"
      gridTemplateColumns={3}
      gap={4}
    >
      {Array.from({ length: 9 }, (_, index) => index + 1).map((value) => (
        <Box
          key={value}
          py={4}
          px={6}
          borderRadius="md"
          bg="fuchsia-500"
          color="white"
          display="flex"
          alignItems="center"
          justifyContent="center"
          fontSize="2xl"
          fontWeight="extrabold"
        />
      ))}
    </Box>
  </template>
  <Box display="grid" gap={4} gridTemplateColumns={3}>
    {/* ... */}
  </Box>
</>
```

            textAlign="center"
            bg="rose-500"

            py={4}

          </Box>
          <Box
            display="table-cell"

            Cell 2

            display="table-cell"

            bg="rose-500"

        <Box display="table-row">

## Responsive

To control the display property of an element at a specific breakpoint, use responsive object notation. For example, adding the property `display={{ md: "none" }}` to an element would apply the `display="none"` utility at medium screen sizes and above.

```jsx
<Box display={{ md: 'none' }} />
```

For more information about xstyled's responsive design features, check out [Responsive Design](@TODO-link) documentation.
