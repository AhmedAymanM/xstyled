# Position

Utilities for controlling how an element is positioned in the DOM.

| React props        | CSS Properties       |
| ------------------ | -------------------- |
| `position=[value]` | `position: [value];` |

## Static

Use `position="static"` to position an element according to the normal flow of the document.

Any offsets will be ignored and the element will not act as a position reference for absolutely positioned children.

```jsx preview
<>
  <template preview>
    <Box position="relative" bg="fuchsia-200" p={6}>
      <Box
        position="static"
        h={32}
        bg="fuchsia-300"
        p={4}
        color="fuchsia-600"
        fontWeight="medium"
      >
        <p>Static parent</p>
        <Box
          position="absolute"
          bottom={0}
          left={0}
          bg="fuchsia-500"
          p={4}
          color="fuchsia-50"
        >
          <p>Absolute child</p>
        </Box>
      </Box>
    </Box>
  </template>
  <Box position="static">
    <p>Static parent</p>
    <Box position="absolute" bottom={0} left={0}>
      <p>Absolute child</p>
    </Box>
  </Box>
</>
```

## Relative

Use `position="relative"` to position an element according to the normal flow of the document.

Offsets are calculated relative to the element's normal position and the element will act as a position reference for absolutely positioned children.

```jsx preview
<>
  <template preview>
    <Box position="relative" bg="red-200" p={6}>
      <Box
        position="relative"
        h={32}
        bg="red-300"
        p={4}
        color="red-600"
        fontWeight="medium"
      >
        <p>Relative parent</p>
        <Box
          position="absolute"
          bottom={0}
          left={0}
          bg="red-500"
          p={4}
          color="red-50"
        >
          <p>Absolute child</p>
        </Box>
      </Box>
    </Box>
  </template>
  <Box position="relative">
    <p>Static parent</p>
    <Box position="absolute" bottom={0} left={0}>
      <p>Absolute child</p>
    </Box>
  </Box>
</>
```

## Absolute

Use `position="absolute"` to position an element outside of the normal flow of the document, causing neighboring elements to act as if the element doesn't exist.

Offsets are calculated relative to the nearest parent that has a position other than `position="static"`, and the element will act as a position reference for other absolutely positioned children.

```jsx preview
<>
  <template preview>
    <Box spaceY={6}>
      <div>
        <Box as="p" fontSize="sm" color="indigo-900" mb={2}>
          With static positioning
        </Box>
        <Box
          position="relative"
          bg="indigo-200"
          px={6}
          pt={2}
          pb={6}
          fontWeight="medium"
        >
          <Box as="p" mb={2} color="indigo-600">
            Relative parent
          </Box>
          <Box position="static" bg="indigo-300" p={4} color="indigo-700">
            <Box as="p" mb={2}>
              Static parent
            </Box>
            <Box
              position="static"
              bottom={0}
              left={0}
              bg="indigo-500"
              p={4}
              color="indigo-50"
              display="inline-block"
            >
              <p>Static child</p>
            </Box>
            <Box
              bg="indigo-200"
              p={4}
              color="indigo-600"
              display="inline-block"
            >
              <p>Static sibling</p>
            </Box>
          </Box>
        </Box>
      </div>

      <div>
        <Box as="p" fontSize="sm" color="indigo-900" mb={2}>
          With absolute positioning
        </Box>
        <Box
          position="relative"
          bg="indigo-200"
          px={6}
          pt={2}
          pb={6}
          fontWeight="medium"
        >
          <Box as="p" mb={2} color="indigo-600">
            Relative parent
          </Box>
          <Box position="static" bg="indigo-300" p={4} color="indigo-700">
            <Box as="p" mb={2}>
              Static parent
            </Box>
            <Box
              position="absolute"
              top={0}
              right={0}
              bg="indigo-500"
              p={4}
              color="indigo-50"
              display="inline-block"
            >
              <p>Absolute child</p>
            </Box>
            <Box
              bg="indigo-200"
              p={4}
              color="indigo-600"
              display="inline-block"
            >
              <p>Static sibling</p>
            </Box>
          </Box>
        </Box>
      </div>
    </Box>
  </template>
  <Box position="static">
    {/* Static parent */}
    <Box position="static">
      <p>Static child</p>
    </Box>
    <Box display="inline-block">
      <p>Static sibling</p>
    </Box>
    {/* Static parent */}
    <Box position="absolute">
      <p>Absolute child</p>
    </Box>
    <Box display="inline-block">
      <p>Static sibling</p>
    </Box>
  </Box>
</>
```

## Responsive

To change how an element is positioned only at a specific breakpoint, use responsive object notation. For example, adding the property `position={{ md: "absolute" }}` to an element would apply the `position="absolute"` utility at medium screen sizes and above.

```jsx
<Box position={{ md: 'absolute' }} />
```
