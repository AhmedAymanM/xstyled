# Min-Width

Utilities for setting the minimum width of an element

| React props        | CSS Properties        |
| ------------------ | --------------------- |
| `minWidth=[value]` | `min-width: [value];` |

## Scaled Min-Width

All values specified in the `sizes` theme section are automatically applied. Note there is a little difference for scale `1`, you have to specify `1s`.

```jsx preview
<>
  <template preview>
    <Box display="flex" flexDirection="column" spaceY={4} mx="auto" w={64}>
      {['1s', 8, 12, 16, 24].map(v => (
        <Box key={v} display="flex" alignItems="center" spaceX={4}>
          <Box
            as="p"
            fontSize="sm"
            color="light-blue-600"
            w={24}
            textAlign="right"
          >
            minWidth={v}
          </Box>
          <Box
            h={8}
            minWidth={v}
            borderRadius="md"
            bg="light-blue-500"
            color="white"
            fontSize="2xl"
            fontWeight="extrabold"
          />
        </Box>
      ))}
    </Box>
  </template>
  <Box>
    <Box minWidth="1s" w={0} />
    <Box minWidth={8} w={0} />
    <Box minWidth={12} w={0} />
    <Box minWidth={16} w={0} />
    <Box minWidth={24} w={0} />
  </Box>
</>
```

## Fixed Min-Width

Any valid value is accepted in width, numbers are converted to `px`, other units have to be specified.

```jsx preview
<>
  <template preview>
    <Box display="flex" flexDirection="column" spaceY={4} mx="auto" w={64}>
      {[123, '12px', '4rem', '3ex'].map(v => (
        <Box key={v} display="flex" alignItems="center" spaceX={4}>
          <Box
            as="p"
            fontSize="sm"
            color="emerald-600"
            w={24}
            textAlign="right"
          >
            minWidth={v}
          </Box>
          <Box
            h={8}
            minWidth={v}
            borderRadius="md"
            bg="emerald-500"
            color="white"
            fontSize="2xl"
            fontWeight="extrabold"
          />
        </Box>
      ))}
    </Box>
  </template>
  <Box>
    <Box minWidth={123} w={0} />
    <Box minWidth="12px" w={0} />
    <Box minWidth="4rem" w={0} />
    <Box minWidth="3ex" w={0} />
  </Box>
</>
```

## Fluid Min-Width

Values from `0` to `1` are converted into percentages. As a fraction or a number, both are an expression. Of course specifying `[value]%` is also possible.

```jsx preview
<>
  <template preview>
    <Box spaceY={6}>
      {[
        [['1/2', 0.5], 0.1],
        [['0.5', 0.5], 0.7],
      ].map((v, i) => (
        <Box
          key={i}
          display="flex"
          overflow="hidden"
          borderRadius="md"
          color="white"
          fontFamily="mono"
          bg="gray-200"
        >
          <Box w={v[0][1]} px={2} py={4} bg="rose-400" textAlign="center">
            minWidth={v[0][0]}
          </Box>
          {v[1] > v[0][1] && (
            <Box
              w={v[1] - v[0][1]}
              px={2}
              py={4}
              bg="rose-500"
              textAlign="center"
            >
              {`0.2`}
            </Box>
          )}
        </Box>
      ))}
    </Box>
  </template>
  <Box minWidth={1 / 2} w={0.1}>
    minWidth=1/2
  </Box>
  <Box minWidth={0.5} w={0.7}>
    minWidth=0.5
  </Box>
</>
```

## Responsive

To control the margin of an element at a specific breakpoint, use responsive object notation. For example, adding the property `minWidth={{ md: 1 }}` to an element would apply the `minWidth={1}` utility at medium screen sizes and above.

```jsx
<Box minWidth={{ xs: 1 / 2, md: 1 }} />
```

For more information about xstyled's responsive design features, check out [Responsive Design](@TODO-link) documentation.

## Customizing

### Sizes scale

f you'd like to customize your values for width, height, min-width, min-height, max-width, max-height, all at once, use the `theme.sizes` section of your theme.
