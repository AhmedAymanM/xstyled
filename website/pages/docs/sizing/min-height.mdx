# Min-Height

Utilities for setting the minimum height of an element

| React props         | CSS Properties         |
| ------------------- | ---------------------- |
| `minHeight=[value]` | `min-height: [value];` |

## Scaled Min-Height

All values specified in the `sizes` theme section are automatically applied. Note there is a little difference for scale `1`, you have to specify `1s`.

```jsx preview
<>
  <template preview>
    <Box
      display="flex"
      justifyContent="space-around"
      alignItems="flex-end"
      mx="auto"
    >
      {['1s', 8, 12, 16, 24].map(v => (
        <Box
          key={v}
          spaceY={4}
          display="flex"
          flexDirection="column"
          alignItems="center"
        >
          <Box
            minHeight={v}
            w={8}
            borderRadius="md"
            bg="light-blue-500"
            color="white"
            fontSize="2xl"
            fontWeight="extrabold"
          />
          <Box as="p" fontSize="sm" color="light-blue-600" textAlign="center">
            minHeight={v}
          </Box>
        </Box>
      ))}
    </Box>
  </template>
  <Box>
    <Box minHeight="1s" h={0} />
    <Box minHeight={8} h={0} />
    <Box minHeight={12} h={0} />
    <Box minHeight={16} h={0} />
    <Box minHeight={24} h={0} />
  </Box>
</>
```

## Fixed Min-Height

Any valid value is accepted in width, numbers are converted to `px`, other units have to be specified.

```jsx preview
<>
  <template preview>
    <Box
      display="flex"
      justifyContent="space-around"
      alignItems="flex-end"
      mx="auto"
    >
      {[123, '12px', '4rem', '3ex'].map(v => (
        <Box
          key={v}
          spaceY={4}
          display="flex"
          flexDirection="column"
          alignItems="center"
        >
          <Box
            w={8}
            minHeight={v}
            borderRadius="md"
            bg="emerald-500"
            color="white"
            fontSize="2xl"
            fontWeight="extrabold"
          />
          <Box as="p" fontSize="sm" color="emerald-600" textAlign="center">
            minHeight={v}
          </Box>
        </Box>
      ))}
    </Box>
  </template>
  <Box>
    <Box minHeight={123} h={0} />
    <Box minHeight="12px" h={0} />
    <Box minHeight="4rem" h={0} />
    <Box minHeight="3ex" h={0} />
  </Box>
</>
```

## Fluid Min-Height

Values from `0` to `1` are converted into percentages. As a fraction or a number, both are an expression. Of course specifying `[value]%` is also possible.

```jsx preview
<>
  <template preview>
    <Box
      display="flex"
      justifyContent="space-around"
      alignItems="flex-end"
      mx="auto"
    >
      {[
        [['1/2', 0.5], 0.1],
        [['0.5', 0.5], 0.7],
      ].map((v, i) => (
        <Box
          key={i}
          overflow="hidden"
          borderRadius="md"
          color="white"
          fontFamily="mono"
          h={48}
          display="flex"
          flexDirection="column"
          justifyContent="flex-end"
          bg="gray-200"
        >
          {v[1] > v[0][1] && (
            <Box
              h={v[1] - v[0][1]}
              px={2}
              py={4}
              bg="rose-500"
              display="flex"
              alignItems="center"
              justifyContent="center"
            >
              {`0.2`}
            </Box>
          )}
          <Box
            h={v[0][1]}
            px={2}
            py={4}
            bg="rose-400"
            display="flex"
            alignItems="center"
            justifyContent="center"
          >
            minHeight={v[0][0]}
          </Box>
        </Box>
      ))}
    </Box>
  </template>
  <Box minHeight={1 / 2} w={0.1}>
    minHeight=1/2
  </Box>
  <Box minHeight={0.5} w={0.7}>
    minHeight=0.5
  </Box>
</>
```

## Responsive

To control the margin of an element at a specific breakpoint, use responsive object notation. For example, adding the property `minHeight={{ md: 1 }}` to an element would apply the `minHeight={1}` utility at medium screen sizes and above.

```jsx
<Box minHeight={{ xs: 1 / 2, md: 1 }} />
```

For more information about xstyled's responsive design features, check out [Responsive Design](@TODO-link) documentation.

## Customizing

### Sizes scale

f you'd like to customize your values for width, height, min-width, min-height, max-width, max-height, all at once, use the `theme.sizes` section of your theme.

```diffjs
  // theme.js
  export const theme = {
    sizes: {
+     sm: '8px',
+     md: '16px',
+     lg: '24px',
+     xl: '48px',
    },
  }
```
