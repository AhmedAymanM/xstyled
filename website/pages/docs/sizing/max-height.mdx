# Max-Height

Utilities for setting the maximum height of an element

| React props         | CSS Properties         |
| ------------------- | ---------------------- |
| `maxHeight=[value]` | `max-height: [value];` |

## Scaled Max-Height

All values specified in the `sizes` theme section are automatically applied. Note there is a little difference for scale `1`, you have to specify `1s`.

```jsx preview
<>
  <template preview>
    <Box
      display="flex"
      justifyContent="space-around"
      alignItems="flex-end"
      mx="auto"
    >
      {['1s', 8, 12, 16, 24].map(v => (
        <Box
          key={v}
          display="flex"
          flexDirection="column"
          alignItems="center"
          position="relative"
          spaceY={4}
        >
          <Box
            h={32}
            bg="gray-200"
            borderRadius="md"
            display="flex"
            alignItems="flex-end"
          >
            <Box
              w={8}
              maxHeight={v}
              h={32}
              borderRadius="md"
              bg="light-blue-500"
              color="white"
              fontSize="2xl"
              fontWeight="extrabold"
            />
          </Box>
          <Box as="p" fontSize="sm" color="light-blue-600" textAlign="right">
            maxHeight={v}
          </Box>
        </Box>
      ))}
    </Box>
  </template>
  <Box>
    <Box maxHeight="1s" h={32} />
    <Box maxHeight={8} h={32} />
    <Box maxHeight={12} h={32} />
    <Box maxHeight={16} h={32} />
    <Box maxHeight={24} h={32} />
  </Box>
</>
```

## Fixed Max-Height

Any valid value is accepted in width, numbers are converted to `px`, other units have to be specified.

```jsx preview
<>
  <template preview>
    <Box
      display="flex"
      justifyContent="space-around"
      alignItems="flex-end"
      mx="auto"
    >
      {[110, '12px', '4rem', '3ex'].map(v => (
        <Box
          key={v}
          display="flex"
          flexDirection="column"
          alignItems="center"
          position="relative"
          spaceY={4}
        >
          <Box
            h={32}
            bg="gray-200"
            borderRadius="md"
            display="flex"
            alignItems="flex-end"
          >
            <Box
              w={8}
              maxHeight={v}
              h={32}
              borderRadius="md"
              bg="emerald-500"
              color="white"
              fontSize="2xl"
              fontWeight="extrabold"
            />
          </Box>
          <Box
            as="p"
            fontSize="sm"
            color="emerald-600"
            w={24}
            textAlign="right"
          >
            maxHeight={v}
          </Box>
        </Box>
      ))}
    </Box>
  </template>
  <Box>
    <Box maxHeight={110} h="999px" />
    <Box maxHeight="12px" h="999px" />
    <Box maxHeight="4rem" h="999px" />
    <Box maxHeight="3ex" h="999px" />
  </Box>
</>
```

## Fluid Max-Height

Values from `0` to `1` are converted into percentages. As a fraction or a number, both are an expression. Of course specifying `[value]%` is also possible.

```jsx preview
<>
  <template preview>
    <Box
      display="flex"
      justifyContent="space-around"
      alignItems="flex-end"
      mx="auto"
    >
      <Box
        overflow="hidden"
        borderRadius="md"
        color="white"
        fontFamily="mono"
        h={48}
        display="flex"
        flexDirection="column"
        justifyContent="flex-end"
        bg="gray-200"
      >
        <Box
          h={0.3}
          px={2}
          py={4}
          bg="rose-500"
          display="flex"
          alignItems="center"
          justifyContent="center"
        >
          0.3
        </Box>
      </Box>
      <Box
        overflow="hidden"
        borderRadius="md"
        color="white"
        fontFamily="mono"
        h={48}
        display="flex"
        flexDirection="column"
        justifyContent="flex-end"
        bg="gray-200"
      >
        <Box
          h={0.2}
          px={2}
          py={4}
          bg="gray-400"
          display="flex"
          alignItems="center"
          justifyContent="center"
        >
          0.2
        </Box>
        <Box
          h={0.5}
          px={2}
          py={4}
          bg="rose-500"
          display="flex"
          alignItems="center"
          justifyContent="center"
        >
          0.5
        </Box>
      </Box>
    </Box>
  </template>
  <Box maxHeight={1 / 2} h={0.3}>
    maxHeight=1/2
  </Box>
  <Box maxHeight={0.5} h={0.7}>
    maxHeight=0.5
  </Box>
</>
```

## Responsive

To control the margin of an element at a specific breakpoint, use responsive object notation. For example, adding the property `maxHeight={{ md: 1 }}` to an element would apply the `maxHeight={1}` utility at medium screen sizes and above.

```jsx
<Box maxHeight={{ xs: 1 / 2, md: 1 }} />
```

For more information about xstyled's responsive design features, check out [Responsive Design](@TODO-link) documentation.

## Customizing

### Sizes scale

f you'd like to customize your values for width, height, min-width, min-height, max-width, max-height, all at once, use the `theme.sizes` section of your theme.
