# Grid Template Areas

Utilities for specifying the template areas in a grid layout.

| React props                 | CSS Properties                  |
| --------------------------- | ------------------------------- |
| `gridTemplateAreas=[value]` | `grid-template-areas: [value];` |
| `gridArea=[value]`          | `grid-area: [value];`           |

## Usage

Use the `gridTemplateAreas={value}` utilities to specify area positions in grids.

```jsx preview
<>
  <template preview>
    <Box
      display="grid"
      gridTemplateColumns={3}
      gap={4}
      gridTemplateAreas='"a a a" "b b c"'
    >
      <Box
        gridArea="a"
        h={12}
        borderRadius="md"
        bg="fuchsia-500"
        color="white"
        display="flex"
        alignItems="center"
        justifyContent="center"
        fontSize="2xl"
        fontWeight="extrabold"
      >
        A
      </Box>
      <Box
        gridArea="b"
        h={12}
        borderRadius="md"
        bg="fuchsia-500"
        color="white"
        display="flex"
        alignItems="center"
        justifyContent="center"
        fontSize="2xl"
        fontWeight="extrabold"
      >
        B
      </Box>
      <Box
        gridArea="c"
        h={12}
        borderRadius="md"
        bg="fuchsia-500"
        color="white"
        display="flex"
        alignItems="center"
        justifyContent="center"
        fontSize="2xl"
        fontWeight="extrabold"
      >
        C
      </Box>
    </Box>
  </template>
  <Box
    display="grid"
    gridTemplateColumns={3}
    gridTemplateAreas='"a a a" "b b c"'
  >
    <Box gridArea="a">A</Box>
    <Box gridArea="b">B</Box>
    <Box gridArea="c">C</Box>
  </Box>
</>
```

## Responsive

To control the grid template areas at a specific breakpoint, use responsive object notation. For example, adding the property `gridTemplateAreas={{ md: '"header header"' }}` to an element would apply the `gridTemplateAreas='"header header"'` utility at medium screen sizes and above.

```jsx
<Box display="grid" gridTemplateAreas={{ md: '"header header"' }}>
  {/* ... */}
</Box>
```
