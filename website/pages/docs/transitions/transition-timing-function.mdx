# Transition Timing Function

Utilities for controlling the easing of CSS transitions.

| React props                        | CSS Properties                         |
| ---------------------------------- | -------------------------------------- |
| `transitionTimingFunction=[value]` | `transition-timing-function: [value];` |

## Usage

Use `transitionTimingFunction={timingFunction}` utilities to control an element's easing curve.

```jsx preview
<>
  <template preview>
    <Box display="flex" justifyContent="space-around" alignItems="center">
      <Box
        as="button"
        fontWeight="medium"
        transition
        transitionTimingFunction="ease-in"
        transitionDuration={700}
        color="white"
        bg="blue-600"
        hoverBg="red-600"
        py={3}
        px={6}
        borderRadius="md"
        border={0}
      >
        Hover me
      </Box>
      <Box
        as="button"
        fontWeight="medium"
        transition
        transitionTimingFunction="ease-out"
        transitionDuration={700}
        color="white"
        bg="blue-600"
        hoverBg="red-600"
        py={3}
        px={6}
        borderRadius="md"
        border={0}
      >
        Hover me
      </Box>
      <Box
        as="button"
        fontWeight="medium"
        transition
        transitionTimingFunction="ease-in-out"
        transitionDuration={700}
        color="white"
        bg="blue-600"
        hoverBg="red-600"
        py={3}
        px={6}
        borderRadius="md"
        border={0}
      >
        Hover me
      </Box>
    </Box>
  </template>
  <Box transition transitionDuration={700} transitionTimingFunction="ease-in">
    Hover me
  </Box>
  <Box transition transitionDuration={700} transitionTimingFunction="ease-out">
    Hover me
  </Box>
  <Box
    transition
    transitionDuration={700}
    transitionTimingFunction="ease-in-out"
  >
    Hover me
  </Box>
</>
```

## Responsive

To control an element's transition-timing-function at a specific breakpoint, use responsive object notation. For example, adding the property `transitionTimingFunction={{ md: "ease-in" }}` to an element would apply the `transitionTimingFunction="ease-in"` utility at medium screen sizes and above.

```jsx
<Box transitionTimingFunction={{ md: 'ease-in' }} />
```

For more information about xstyled's responsive design features, check out [Responsive Design](@TODO-link) documentation.

## Customizing

### Timing Functions

If you'd like to customize your values for all timing functions, use the `theme.timingFunctions` section of your theme.

```diffjs
  // theme.js

  export const theme = {
    timingFunctions: {
      // ...
      'ease-in': 'cubic-bezier(0.4, 0, 1, 1)',
      'ease-out': 'cubic-bezier(0, 0, 0.2, 1)',
      'ease-in-out': 'cubic-bezier(0.4, 0, 0.2, 1)',
+     'ease-in-expo': 'cubic-bezier(0.7, 0, 0.84, 0)',
    },
  }
```
