# Transition

Utilities for controlling transitions.

| React props          | CSS Properties         |
| -------------------- | ---------------------- |
| `transition=[value]` | `transition: [value];` |

## Usage

Use `transition={transition}` to specify transitions.

```jsx preview
<>
  <template preview>
    <Box textAlign="center">
      <Box
        as="button"
        fontWeight="medium"
        transition
        transitionDuration={500}
        color="white"
        bg="blue-600"
        hoverBg="red-600"
        py={3}
        px={6}
        borderRadius="md"
        border={0}
      >
        Hover me
      </Box>
    </Box>
  </template>
  <Box
    as="button"
    transition
    transitionDuration={500}
    color="white"
    bg="blue-600"
    hoverBg="red-600"
    py={3}
    px={6}
    borderRadius="md"
  >
    Hover me
  </Box>
</>
```

## Prefers-reduced-motion

You can conditionally apply animations and transitions using the `motionSafe` and `motionReduce` variants:

```jsx
<Box as="button" hoverColor="red" transition motionReduceTransition="none">
  Hover me
</Box>
```

## Responsive

To control the transitions on an element at a specific breakpoint, use responsive object notation. For example, adding the property `transition={{ md: "all" }}` to an element would apply the `transition="all"` utility at medium screen sizes and above.

```jsx
<Box transition={{ md: 'all' }} />
```

For more information about xstyled's responsive design features, check out [Responsive Design](@TODO-link) documentation.

### Transitions

If you'd like to customize your values for transitions, use the `theme.transitions` section of your theme.

```diffjs
  // theme.js

  export const theme = {
    transitions: {
      // ...
+     light: 'color 300ms ease-in-out'
    },
  }
```
