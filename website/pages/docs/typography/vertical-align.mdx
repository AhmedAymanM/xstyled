# Vertical Alignment

Utilities for controlling the vertical alignment of an inline or table-cell box.

| React props             | CSS Properties            |
| ----------------------- | ------------------------- |
| `verticalAlign=[value]` | `verticalAlign: [value];` |

## Baseline

Use `verticalAlign="baseline"` to align the baseline of an element with the baseline of its parent.

```jsx preview
<>
  <template preview>
    <Box lineHeight="none" position="relative">
      <Box
        as="span"
        w={0}
        h={8}
        display="inline-block"
        verticalAlign="baseline"
      >
        <Box
          as="span"
          position="absolute"
          top={0}
          borderColor="light-blue-300"
          borderStyle="dashed"
          borderWidth="1 0"
          w={1}
          h={8}
        />
        <Box
          as="span"
          position="absolute"
          top={0}
          borderColor="light-blue-300"
          borderStyle="dashed"
          borderWidth="1 0"
          w={1}
          h={4}
        />
      </Box>
      <Box
        as="span"
        position="relative"
        zIndex={10}
        color="light-blue-600"
        fontWeight="medium"
      >
        Computers have lots of memory but no imagination.
      </Box>
    </Box>
  </template>
  <Box as="span" display="inline-block" verticalAlign="baseline">
    ...
  </Box>
</>
```

## Responsive

To control the vertical alignment of an element at a specific breakpoint, use responsive object notation. For example, adding the property `verticalAlign={{ md: "text-bottom" }}` to an element would apply the `verticalAlign="text-bottom"` utility at medium screen sizes and above.

```jsx
<Box verticalAlign={{ md: 'text-bottom' }}>{/* ... */}</Box>
```
