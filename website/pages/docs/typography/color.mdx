# Text Color

Utilities for controlling the text color of an element.

| React props     | CSS Properties    |
| --------------- | ----------------- |
| `color=[value]` | `color: [value];` |

## Usage

Control the text color of an element using the `color={color}` utilities.

```jsx preview
<>
  <template preview>
    <Box spaceY={8}>
      {['red-500', 'emerald-700', '#f59e0b', 'rgba(76, 29, 149, 0.6)'].map(
        color => (
          <Box key={color} as="dl" color="gray-500" overflow="hidden">
            <Box
              as="dt"
              flexShrink={0}
              fontSize="sm"
              opacity={0.8}
              fontFamily="mono"
              mb={1}
            >
              color={color}
            </Box>
            <Box as="dd" fontSize="xl" m={0}>
              <Box color={color}>
                Computers have lots of memory but no imagination.
              </Box>
            </Box>
          </Box>
        ),
      )}
    </Box>
  </template>
  <Box color="red-500">Computers have ...</Box>
  <Box color="emerald-700">Computers have ...</Box>
  <Box color="#f59e0b">Computers have ...</Box>
  <Box color="rgba(4, 120, 87, 0.5)">Computers have ...</Box>
</>
```

## Hover

To control the text color of an element on hover, use `hoverColor={color}` utility.

```jsx preview
<>
  <template preview>
    <Box color="red-400" hoverColor="red-800">
      Computers have lots of memory but no imagination.
    </Box>
  </template>
  <Box color="red-400" hoverColor="red-800">
    Computers have lots of memory but no imagination.
  </Box>
</>
```

## Focus

To control the text color of an element on hover, use `focusColor={color}` utility.

```jsx preview
<>
  <template preview>
    <Box
      as="input"
      defaultValue="Something"
      color="red-600"
      focusColor="red-800"
      borderRadius="md"
      border={1}
      borderColor="red-600"
      p={2}
      focusOutline="none"
    />
  </template>
  <Box
    as="input"
    defaultValue="Something"
    color="red-600"
    focusColor="red-800"
  />
</>
```

## Placeholder

To control the placeholder color of an element, use `placeholderColor={color}` utility.

```jsx preview
<>
  <template preview>
    <Box
      as="input"
      placeholder="Type something..."
      color="red-600"
      placeholderColor="red-300"
      borderRadius="md"
      border={1}
      borderColor="red-600"
      p={2}
      focusOutline="none"
    />
  </template>
  <Box
    as="input"
    placeholder="Type something..."
    color="red-600"
    placeholderColor="red-300"
  />
</>
```

## Responsive

To control the space between elements at a specific breakpoint, use responsive object notation. For example, adding the property `color={{ md: "red-600" }}` to an element would apply the `color="red-600"` utility at medium screen sizes and above.

```jsx
<Box color={{ md: 'red-600' }}>{/* ... */}</Box>
```

For more information about xstyled's responsive design features, check out [Responsive Design](@TODO-link) documentation.

## Customizing

### Colors

If you'd like to customize your values for all colors, use the `theme.colors` section of your theme.

```diffjs
  // theme.js
  import { th } from '@xstyled/styled-components'

  export const theme = {
    colors: {
      // ...
+     'deep-blue': '#00008b',
+     'primary': th.color('red-600'),
    },
  }
```
